this.index == button.length - 1 && (box.style.cssText = '');

cssText 属性

元素节点对象的 style 对象，有一个 cssText 属性，可以读写或删除整个样式。

操作 CSS 样式的另外几种方式：

  1. 使用网页元素节点的 getAttribute 方法、setAttribute 方法和 removeAttribute 方法，直接读写或删除网页元素的 style 属性。

  2. 每一个网页元素对应一个 DOM 节点对象，这个对象的 style 属性可以直接操作，用来读写行内 CSS 样式。

    style 属性的值是一个对象，这个对象所包含的属性与 CSS 规则一一对应。属性若使用「点符号」，则名字需要改写，改写规则就是将横杠从 CSS 属性名中去除，然后横杠后的第一个字母大写，如：background-color 写成 backgroundColor。

  3. style 对象的 getPropertyValue(propertyName) 方法、setProperty(propertyName, value) 方法和 removeProperty(propertyName) 方法，用来读写行内 CSS 规则。

&& 符号

  • a && b：如果 a 返回 true，则执行 b；如果 a 返回 false，则不执行 b；

  • a || b：如果 a 返回 true，则不执行 b；如果 a 返回 false，则执行 b；

故，this.index == button.length - 1 && (box.style.cssText = ''); 等同于 if (this.index == button.length - 1) { box.style.cssText = ''; }。若 this.index 的值为最后一个索引（即重置按钮），则清空样式。

oBtn[i].index = i;
